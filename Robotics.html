CS 3331-Robotics

Class information

Honestly a fun class if you’re interested in learning how to build and program a robot on an introductory level. The course requires a microprocessor and a robotics kit you can get off Amazon, and the example code for the projects is well documented and easy to understand. The group project is really fun and “hands-on”. Our project was a self-driving smart car that followed a path created by black tape, where the light and camera sensor picked on the path navigated by the contrast against a white platform. No matter where the tape was posted, the robot followed. The professor takes you through a step-by-step instruction, from the history of robotics to the modern ones. I would recommend this course for anyone willing to dip their toes into the water of automation.
